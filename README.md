# ğŸ“ Overview

The **Student Academic Hub** is a complete academic management system developed entirely in **C language**.  
It helps students manage:

- Academic results  
- GPA & CGPA tracking  
- Tasks & to-do lists  
- Study routines  
- Book lists  
- Login/Registration  

All data is stored through **file handling**, ensuring long-term record keeping.
